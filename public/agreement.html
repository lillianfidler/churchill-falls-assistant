Cost Controls & Safeguards

  • Rate limiting: 20 requests per hour per IP address, preventing abuse or bot traffic
  • Response caching: Identical questions within a 1-hour window are served from cache at zero API cost
  • Prompt caching: System documents are cached at the API level, reducing input token costs by ~90% on repeated requests
  • Smart model selection: The more expensive Sonnet model is only used for Deep Research; the more economical Haiku model handles Voice and Fast modes at one-third the cost
  • Tool call limits: Deep Research is capped at 5 document retrievals per question, reducing context accumulation while maintaining thorough answers
  • Monthly voice tracking: ElevenLabs character usage is monitored in real-time; the system gracefully falls back to text-only if the monthly voice allowance is approached
  • Spending alerts: Email notifications are triggered when API costs approach set thresholds
  • Usage dashboard: A live statistics dashboard tracks all questions, response times, mode usage, and system health
